# Test Command: Help

`pilum add "Buy milk"` adds a new pending task to the task list:

```console
$ pilum add "Buy milk"
Created task 1: Buy milk

```

`pilum add "Buy eggs"` after `pilum add "Buy milk` adds a second pending task to the task list:

```console
$ pilum add "Buy milk"
Created task 1: Buy milk

$ pilum add "Buy eggs"
Created task 2: Buy eggs

```
